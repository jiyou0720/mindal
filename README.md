MindBridge: AI 기반 심리상담 플랫폼 상세 기획 (최종 업데이트)
MindBridge는 심리 테스트와 AI 챗봇을 통해 사용자가 자신의 감정을 이해하고, 맞춤형 심리상담을 받을 수 있도록 돕는 웹 기반 플랫폼입니다. 심리 상담 및 마인드 세팅을 지원하며, 비슷한 고민을 가진 사용자들끼리 소통할 수 있는 커뮤니티 기능도 제공할 예정입니다.

핵심 가치
익명성 보장: AI 챗봇 상담, 게시판, 댓글 등 모든 서비스에서 사용자의 익명성을 철저히 보장합니다.

심리적 안정 제공: 위로, 공감, 인지행동치료(CBT) 등 다양한 상담 스타일을 선택할 수 있어 사용자에게 최적화된 심리적 안정감을 제공합니다.

사용자 맞춤형 경험: 심리 테스트 결과를 기반으로 AI 챗봇 상담이 연동되어 개인화된 경험을 제공합니다.

주요 기능
1. 심리 테스트
성격 유형 테스트: 다양한 성격 유형 검사를 통해 사용자의 기본적인 성향을 파악합니다.

감정 상태 진단 테스트: 현재 사용자의 감정 상태를 진단하고 구체적인 어려움을 식별합니다.

테스트 결과 요약 및 그래픽 제공: 직관적인 그래픽과 함께 테스트 결과를 요약하여 제공하며, 심리 상태를 쉽게 이해할 수 있도록 돕습니다.

2. 챗봇 상담
심리 테스트 결과 기반 챗봇 연동: 심리 테스트 결과에 맞춰 AI 챗봇이 개인화된 상담을 제공합니다.

상담 스타일 선택:

일반 공감형: 사용자의 감정에 공감하고 위로를 제공하는 상담 스타일입니다.

인지행동치료(CBT) 기반: 문제의 원인이 되는 비합리적인 생각을 탐색하고 수정하도록 돕습니다.

솔루션 중심 상담: 구체적인 문제 해결 방안을 함께 모색하고 목표 지향적인 상담을 진행합니다.

대화 기록 저장 및 상담 요약본: 상담 대화 기록을 저장하고, 상담을 마친 후 핵심 내용을 요약한 보고서를 제공하여 사용자가 자신의 변화 과정을 되돌아볼 수 있도록 합니다.

위험 감지 시 관련 정보 자료 제공: 사용자의 심리 상태가 위험하다고 판단될 경우, 자가 진단 자료, 관련 서적 목록, 유관 기관 정보 등 전문가의 도움을 받을 수 있는 정보를 제공합니다.

3. 감정 일기
감정 선택 및 일기 작성: 사용자가 매일 자신의 감정을 선택하고 그에 따른 일기를 자유롭게 작성할 수 있는 페이지를 제공합니다.

4. 커뮤니티
익명/닉네임 기반 소통: 에브리타임(Everytime)과 유사한 형태로, 사용자들이 익명 또는 닉네임(미설정 시 이름)으로 게시글을 작성하고 소통할 수 있는 공간을 제공합니다.

게시글 기능: 추천, 신고 기능을 포함하며, 작성자는 자신의 글에 추천 불가, 신고 버튼 비활성화, 수정 버튼 활성화 기능을 가집니다.

인기 게시글 노출: 일정 수(예: 10개) 이상의 추천을 받은 게시글은 상단에 노출됩니다.

공지사항 게시: 관리자가 작성한 공지사항은 커뮤니티 페이지 상단에 항상 게시됩니다.

5. 감정 카드 페이지
상담 전/후 감정 표현: 상담을 이용한 사용자들이 상담 전후의 감정 변화를 감정 카드를 통해 시각적으로 표현하고 기록할 수 있습니다.

익명 공개: 기록된 감정 카드는 커뮤니티처럼 누구나 볼 수 있지만, 익명으로 공개됩니다.

6. 감정 뱃지 시스템
활동 기반 뱃지 부여: 사용자가 특정 행동(예: 챗봇 상담 5회 완료, 일기 7일 연속 작성, 커뮤니티 게시글 3개 작성, 특정 감정 선택 횟수 등)을 달성할 때마다 **'감정 뱃지'**를 부여합니다.

다양한 뱃지 종류: 긍정적인 감정(예: '행복 탐험가', '희망의 씨앗'), 꾸준한 활동(예: '꾸준함의 아이콘', '마음 지킴이'), 특정 문제 극복(예: '불안 극복 뱃지', '스트레스 해소 뱃지') 등 다양한 종류의 뱃지를 기획합니다.

마이페이지에 뱃지 컬렉션: 사용자의 마이페이지에 획득한 뱃지들을 모아볼 수 있는 컬렉션 기능을 제공하여 성취감과 동기 부여를 높입니다.

사용자 인터페이스
깔끔하고 편안한 분위기의 디자인: 사용자에게 심리적 안정감을 제공하는 미니멀하고 차분한 디자인을 지향합니다.

다크모드/라이트모드 지원: 사용자의 취향과 환경에 따라 테마를 선택할 수 있도록 합니다.

모바일/PC 반응형 웹 지원: 다양한 기기에서 최적화된 사용자 경험을 제공합니다.

기술 스택
Front-End:

HTML / CSS / JavaScript: 웹 페이지의 기본적인 구조, 디자인, 동적인 상호작용 구현.

React + Next.js: 컴포넌트 기반의 효율적인 코드 관리 및 빠른 페이지 전환을 위한 프레임워크 사용.

Recharts: 심리 테스트 결과를 시각적으로 아름다운 그래프로 표현하여 사용자의 이해도를 높임.

Back-End:

Python (Flask): 가볍고 빠르게 API 개발이 가능하며, GPT API 및 자연어 처리(NLP) 기능 연동에 용이.

DB: MySQL (관계형 데이터베이스) / MongoDB (NoSQL 데이터베이스): 사용자 정보, 심리 테스트 결과, 챗봇 상담 로그, 뱃지 획득 정보, 닉네임 변경 내역 등 다양한 형태의 데이터를 안정적으로 저장하고 관리.

GPT API (OpenAI): 고품질의 챗봇 상담 기능을 구현하여 사용자에게 전문적이고 자연스러운 대화 경험 제공.

타겟 사용자
심리 상담에 관심이 있는 일반인: 자신의 감정과 심리 상태를 이해하고 싶은 모든 사용자.

학교•직장에서 스트레스를 겪고 있는 사람: 일상생활 속에서 스트레스와 어려움을 겪는 이들에게 비대면 솔루션 제공.

익명성 있는 안전한 상담 공간을 원하는 사용자: 개인 정보 노출 없이 자유롭게 자신의 고민을 이야기하고 싶은 사용자.

기본 제공 자료 및 정책
회원 가입 시 다음 정보를 기본적으로 제공하고 동의를 얻습니다.

상담 동의서 및 안내문:

AI 상담의 한계 명시: MindBridge는 AI 챗봇을 활용한 보조적인 심리 상담을 제공하며, AI 상담 결과에 지나치게 맹신하지 않도록 안내합니다.

전문가 상담의 중요성 강조: 심각한 심리적 어려움을 겪는 경우 반드시 전문 심리 상담사 또는 의료 전문가를 찾아 직접 상담을 받을 것을 강력히 권고합니다.

상담의 목적, 과정, 비밀 보장 원칙(예외 사항 포함), 이용 약관 등을 명확히 안내합니다.

초기 면담지/설문지: 내담자의 기본 정보(인적 사항), 상담을 받게 된 주된 이유, 현재 겪고 있는 어려움, 과거 경험, 건강 상태 등 상담자가 내담자를 이해하는 데 필요한 기초 자료를 수집합니다.

개인 정보 활용 동의서: 상담 과정에서 수집되는 개인 정보가 어떻게 수집, 이용, 보관, 폐기되는지에 대한 동의를 구합니다. 개인 정보 보호법에 따라 내담자의 권리를 보호하며, 익명 처리된 상담 데이터는 통계 자료로 활용될 수 있음을 명확히 안내합니다.

MindBridge 서비스 소개 자료: MindBridge 웹 플랫폼의 주요 기능, 핵심 가치, 활용 방법 등을 상세히 소개합니다.

페이지 구성
1. 홈 페이지
사이트 접속 시 가장 먼저 보이는 페이지.

MindBridge 프로젝트 소개: 플랫폼의 목표와 핵심 가치를 명확히 제시합니다.

공지사항: 관리자가 작성한 공지사항이 페이지 상단에 게시됩니다.

이용 방법 안내: 심리 테스트 및 챗봇 상담 이용 과정을 간략하게 설명합니다.

로그인/회원가입: 쉽고 빠르게 서비스에 접근할 수 있도록 로그인 및 회원가입 버튼을 제공합니다.

2. 회원가입 페이지
기본 개인 정보 수집: 이름, 성별, 나이, 전화번호 등의 기본적인 개인 정보를 수집합니다.

DB 저장 항목: 회원 이름, 회원 닉네임 (익명성 보장을 위해 사용), 회원 성별, 회원 나이, 회원 전화번호, 회원 학교, 회원 학과, 회원 UID (자동 부여되는 5자리 랜덤 숫자).

개인 정보 동의 및 익명성 보장 명시: 개인 정보 수집 및 활용에 대한 동의를 받고, 익명성은 보장하지만 익명화된 상담 데이터는 통계 자료로 사용될 수 있음을 명확히 안내합니다.

3. 심리 테스트 선택 페이지
제공되는 다양한 심리 테스트 목록을 보여주고 사용자가 원하는 테스트를 선택할 수 있도록 합니다.

4. 테스트 진행 페이지
선택한 심리 테스트의 문항을 순서대로 제시하고 사용자의 답변을 입력받습니다.

5. 결과 페이지
심리 테스트 결과 통계 및 해석: 사용자의 답변을 기반으로 심리 상태를 분석하고 통계 및 해석을 제공합니다.

챗봇 상담 유도 및 연결: 테스트 결과를 바탕으로 AI 챗봇 상담으로 자연스럽게 연결될 수 있도록 유도합니다.

6. 챗봇 상담 페이지
선택한 상담 스타일에 맞춰 AI 챗봇과 실시간으로 대화할 수 있는 인터페이스를 제공합니다.

대화 기록 저장 및 요약본 기능을 포함합니다.

위험 감지 시 관련 정보 제공 기능이 연동됩니다.

챗봇 상담 피드백/별점 시스템: 상담 종료 후 챗봇 상담에 대한 만족도를 평가하고 피드백을 남길 수 있는 기능을 제공합니다. 이 데이터는 개발자 및 관리자만 열람할 수 있으며 AI 챗봇의 성능 개선에 활용됩니다.

7. 감정 일기 페이지
내담자가 이용 가능한 감정 일기 페이지로, 감정 선택 및 일기 작성이 가능합니다.

8. 커뮤니티 페이지
사용자들이 익명 또는 닉네임(닉네임 설정 안 할 시 이름)으로 소통 가능한 게시글 형태의 공간입니다.

추천, 신고 기능이 있으며, 작성자는 추천 불가, 신고 버튼 비활성화, 수정 버튼 활성화 기능을 가집니다.

일정 추천(10개 이상)을 받으면 게시글이 상단에 노출됩니다.

공지사항이 맨 상단에 게시됩니다.

9. 감정 카드 페이지
상담을 한 유저들이 상담 전/후의 감정을 감정 카드를 통해 표현하고 기록하는 페이지입니다.

매일매일 기록 가능하며, 커뮤니티처럼 누구나 볼 수 있지만 익명으로 공개됩니다.

10. 마이페이지
사용자의 개인 정보 및 설정 관리.

닉네임 변경 가능: 사용자가 닉네임을 변경할 수 있도록 하며, 닉네임 변경 내역을 저장합니다.

획득한 감정 뱃지 컬렉션 표시.

지난 심리 테스트 결과 열람.

과거 챗봇 상담 기록 열람 및 요약본 확인.

작성한 감정 일기 및 감정 카드 기록 열람.

11. 문의 / 피드백 페이지
홈 페이지나 챗봇 상담 페이지 등 주요 페이지에서 쉽게 접근할 수 있는 버튼을 제작합니다.

사용자가 서비스 이용 중 겪는 문제나 개선 사항, 제안 등을 자유롭게 문의하고 피드백을 남길 수 있는 공간입니다.

12. 관리자 페이지 - 대시보드
관리자 역할을 가진 사람만 접속 가능합니다. (일부 페이지는 관리자 외 다른 권한을 지닌 사람도 접속 가능)

12-1. 유저 관리: 생성된 모든 계정을 볼 수 있습니다. 개인 정보 보호를 위해 회원 이름, 회원 닉네임, 회원 성별, 회원 나이, 회원 전화번호, 회원 학교, 회원 학과, 회원 UID만 보여주며, 비밀번호는 제외됩니다. 계정 삭제 기능도 포함됩니다.

12-2. 공지사항 관리: 관리자, 운영자가 공지 페이지에서 공지를 작성하면 홈, 게시글 페이지에 노출됩니다. 공지사항은 수정 가능하며, 삭제가 아닌 공개/비공개로 전환할 수 있습니다.

12-3. DB 관리 (상담 기록, 상담 기록 요약본): 관리자, 개발자, 연구자만 접속 가능하며, DB에 저장된 상담 기록 데이터를 볼 수 있습니다.

12-4. 게시글 관리: 관리자, 운영자만 접근 가능하며, 게시글 신고 내역을 확인하고 처리(게시글 하루 정지, N시간 정지, 영구 정지, 게시글 삭제 등)할 수 있습니다.

12-5. 사용자 권한 관리: 관리자만 접근 가능하며, 관리자, 운영자, 개발자, 연구자 등 사용자에게 역할을 부여할 수 있습니다.

12-6. CMS (콘텐츠 관리 시스템): 심리 테스트 문항, 결과 해석, 챗봇 답변 데이터, 위험 감지 시 제공되는 자료 등을 관리자가 쉽게 추가하고 수정할 수 있는 시스템입니다.

12-7. 데이터 분석 및 통계 대시보드: 사용자 활동, 가장 많이 이용하는 테스트, 챗봇 상담 주제, 커뮤니티 인기 게시글 등 서비스 전반의 데이터를 시각적으로 분석하여 서비스 개선 방향 설정에 활용합니다.

12-8. 챗봇 피드백/별점 관리: 챗봇 상담 피드백 및 별점 데이터를 개발자 및 관리자만 열람하고 분석할 수 있는 섹션입니다.

사용자 역할
일반 사용자: 회원가입 시 기본적으로 부여되는 역할입니다. 모든 주요 기능을 이용할 수 있습니다.

게스트: 회원가입 없이 사이트에 접속한 사용자에게 부여되는 역할입니다. 제한된 기능만 이용 가능합니다.

관리자: 모든 관리자 페이지 기능 및 권한 관리, 공지사항 작성/수정, 게시글 신고 처리, DB 접근 권한, CMS, 데이터 분석 대시보드, 챗봇 피드백 관리 권한을 가집니다.

운영자: 공지사항 작성/수정, 게시글 신고 처리 권한을 가집니다.

개발자: DB 접근 권한, 챗봇 피드백 관리 권한을 가집니다.

연구자: DB 접근 권한을 가집니다.

사용자 한명이 여러개의 권한을 지닐 수 있습니다.
페이지 접근을 할때 해당되는 권한이 없으면 접근을 금지합니다.
